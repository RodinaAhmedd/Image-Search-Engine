# Image-Search-Engine
A simple and responsive **Image Search Engine** that allows users to search for images using keywords and view results fetched from the **Unsplash API**.  
The project supports loading more images dynamically with a "Show More" button.

## Technologies Used:

- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript (ES6)
- Unsplash REST API


##  How It Works:

1. User enters a keyword in the search box.
2. JavaScript sends a request to the Unsplash API.
3. Images are fetched and displayed dynamically.
4. Clicking **Show More** loads additional results.
